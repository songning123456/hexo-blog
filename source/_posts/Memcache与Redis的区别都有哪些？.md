# Redis

* 部份存在硬盘上，Redis可以持久化其数据。
* 支持更为丰富的数据类型 ，提供list，set，zset，hash等数据结构的存储。
* 直接自己构建了VM机制，因为一般的系统调用系统函数的话，会浪费一定的时间去移动和请求。
* 最大可以达到512M。
* 速度快。
* 支持master-slave模式的数据备份。

# Memcache

* 数据全部存在内存之中，断电后会挂掉，数据不能超过内存大小。
* 所有的值均是简单的字符串。
* 只有1MB。
* 速度慢。
