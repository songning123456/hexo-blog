# 程序计数器

**私有**

当前线程执行的字节码的行号指示器。

# Java虚拟机栈

**私有**

存放基本数据类型、对象引用和returnAddress类型。

# 本地方法栈

**私有**

为虚拟机使用到的Native方法服务。

# Java堆

**公有**

存放对象的实例，也是GC回收器管理的主要区域。

# 方法区

**公有**

存放已被虚拟机加载的类信息、常量、静态变量、即时编译后的代码等数据。

# 运行时常量池

方法区的一部分，存放编译期生成的各种字面量和符号引用。

# 直接内存

不是虚拟机运行时数据区的一部分，也不是Java虚拟机规范中定义的内存区域，容易引起OOM异常，NIO会调用，不受Java堆大小的限制。
