# 重写(Override)

发生在父子类中；方法名、参数列表必须相同；返回值类型小于等于父类；抛出的异常小于等于父类；访问修饰符大于等于父类；如果父类方法访问修饰符为private则子类中就不是重写。

# 重载(Overload)

发生在同一个类中；方法名必须相同；参数类型不同、个数不同、顺序不同；方法返回值和访问修饰符可以不同；发生在编译时。
